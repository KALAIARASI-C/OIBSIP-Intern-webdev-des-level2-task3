# To-Do List Web Application

## Objective

The objective of this project is to develop a simple **To-Do List Web Application** that allows users to add, view, and delete tasks. The application helps users manage their daily activities efficiently and demonstrates the use of front-end web technologies such as HTML, CSS, and JavaScript to create an interactive web application.

## Steps Performed

1. Designed the basic structure of the webpage using **HTML**.
2. Created a **header section** to display the application title.
3. Developed a **task input form** with fields for task title and description.
4. Styled the application layout using **CSS**, including background image, form card, and task display section.
5. Implemented a **task list section** to display added tasks in a structured format.
6. Used **JavaScript** to create dynamic functionality for adding new tasks to the list.
7. Implemented a **delete button** to remove tasks from the list.
8. Added validation to ensure the user fills in the required fields before saving a task.
9. Tested the application in a browser to ensure smooth functionality.
10. Uploaded the project to **GitHub** for version control and sharing.

## Tools and Technologies Used

* HTML5
* CSS3
* JavaScript
* Visual Studio Code (Code Editor)
* GitHub (Project Hosting)

## Outcome

The final outcome of this project is a **user-friendly To-Do List Web Application** that allows users to add and manage tasks easily. The application provides a simple interface where tasks can be created and deleted dynamically, demonstrating the practical use of JavaScript for interactive web development.
